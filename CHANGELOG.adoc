== 0.1.0 — 2026-04-05

Initial public release.

=== Added

* Core `StringPower.res` library with seven tagged-template modules:
  `Fmt`, `Sql`, `Css`, `React`, `I18n`, `Url`, `Gql`.
* Rust CLI `string-union-gen` that generates `toString` / `fromString`
  converters from `@stringUnion`-annotated ReScript polymorphic variants.
* ReScript test suite (`tests/StringPower_test.res`, 15 assertions).
* Rust unit tests covering the parser and code generator (5 tests).
* Six independent test targets wired through `justfile`:
  structure, unit, end-to-end, property, benchmark, lint.
* `0-AI-MANIFEST.a2ml` for AI-agent onboarding.
* `READINESS.md` documenting Component Readiness Grade B.
* GitHub Actions workflow running all six test targets on every push.
* Contributing guide, code of conduct, and security policy.
